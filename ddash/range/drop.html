    Module ddash.range.drop

    Drops elements from a range

    Example

    assert([1, 2, 3, 4].drop.array == [2, 3, 4]);
assert([1, 2, 3, 4].dropRight.array == [1, 2, 3]);
assert([1, 2, 3, 4].dropWhile!(a => a < 3).array == [3, 4]);
assert([1, 2, 3, 4].dropRightWhile!(a => a > 2).equal([1, 2]));

    Functions

    NameDescription
    drop(range, n) Drops n elements from beginning of range
    dropRight(range, n) Drops n elements from end of range
    dropRightWhile(range) Drops elements from end of range while predicate is true
    dropWhile(range) Drops elements from beginnig of range while predicate is true
